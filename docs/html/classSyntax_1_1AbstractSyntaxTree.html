std::unique_ptr< RegexNode > Syntax::AbstractSyntaxTree::parse_regex ( const std::string &  regex)
private

Parses a regular expression into an AST composed of various node types.

This function interprets the regex pattern and constructs a corresponding tree structure where each node represents a distinct construct in the regex (e.g. a literal character, a character set, etc.) The AST can be used to match stirngs against the regex pattern.

Supported constructs include:

  • Literal characters
  • Character classes (e.g., [a-z], [A-Z], [0-9])
  • Predefined character classes (e.g., \d, \w, \s)
  • Quantifiers (e.g., *, +, ?, {n}, {n,}, {n,m})
  • Alternation (e.g., a|b)
  • Grouping Parentheses (e.g., (a|b))
  • Anchors (e.g., ^, $)
  • Escape characters (e.g., \)
  • Dot (.) to match any character
Parameters
[in]regexThe regular expression string to be parsed into an AST
Returns
A std::unique_ptr to the root node of the constructed AST. If the the regex is invalid or empty, the function returns nullptr.
